Tutorial: ADB via Wi-Fi e scrcpy

1. Conecte via USB

Ative a opção "Depuração USB" no seu Android:
Configurações > Sistema > Opções do desenvolvedor > Depuração USB

Conecte o celular via cabo USB ao seu computador.

2. Verifique a conexão com ADB

Abra um terminal no computador e digite:

adb devices

Confirme a permissão no celular caso solicitado.

3. Ative o modo Wi-Fi

Digite o comando:

adb tcpip 5555

Em seguida descubra o IP do celular:

adb shell ip route

Ou veja nas configurações Wi-Fi do Android.

Agora conecte-se via Wi-Fi:

adb connect 192.168.0.101:5555

Substitua pelo IP do seu celular.

4. Use o scrcpy

Com o celular conectado via Wi-Fi, use o scrcpy:

scrcpy

Para abrir diretamente com tela espelhada sem fio.

Você também pode usar:

scrcpy --bit-rate 2M --max-size 1024

Para conexões Wi-Fi mais leves.

Dicas

  • Mantenha o celular e o PC na mesma rede Wi-Fi.
  • Se a conexão cair, reconecte com adb connect.
  • Use adb disconnect para encerrar a sessão.
  • Você pode automatizar com scripts .sh ou .bat.
